## Course Project
## Course Name: Scalable Data Science

The goal of the project is to process data using the following big data frameworks  and evaluate their performance. 

* MapReduce Framework
* Apache Spark API
* Apache Flink DataSet API 
* Apache Flink Streaming API


**Tools** <br>

* IntelliJ IDEA
* Maven
* Java
